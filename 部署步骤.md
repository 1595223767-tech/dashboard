# 📤 部署到 GitHub Pages（3 步搞定）

## 前提

- 有 GitHub 账号（没有的话 https://github.com/join 注册，1 分钟）
- 能访问 github.com

---

## 方法 A：网页操作（推荐，全程点鼠标）

### 第 1 步：新建仓库

1. 打开 https://github.com/new
2. Repository name 填：`agent-rank-dashboard`（随便起，纯英文）
3. 选 **Public**（Pages 免费版只支持 Public）
4. **不要**勾选 "Add a README file"（我们已经有 README）
5. 点绿色 **Create repository**

### 第 2 步：上传文件

1. 新仓库页面上，点 **"uploading an existing file"**（灰色文字链接）
2. 把这个目录 `看板-github版/` 里的 **4 个文件** 全部拖进浏览器：
   - `index.html`
   - `README.md`
   - `代理价值+协同_数据模板.xlsx`
   - `.nojekyll`（隐藏文件，Mac 按 Cmd+Shift+. 显示）
3. 拉到底部，点 **Commit changes**

### 第 3 步：开启 Pages

1. 仓库页面 → **Settings**（右上角）
2. 左侧菜单 → **Pages**
3. Source 选 **Deploy from a branch**
4. Branch 选 **main** / 目录选 **/ (root)** → 点 **Save**
5. 等 30 秒 - 2 分钟，页面顶部会显示你的链接：
   `https://<你的用户名>.github.io/agent-rank-dashboard/`

搞定！把这个链接发给同事就行。

## 方法 B：命令行（懂 git 的话更快）

```bash
cd "/Users/lixinxuan/WorkBuddy/闭环头客空间/看板-github版"

# 配置 git 身份（首次用）
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"

# 初始化并提交
git init
git add .
git commit -m "init dashboard"
git branch -M main

# 在 github.com 建好空仓库后（不加 README），把下面 URL 换成你的
git remote add origin https://github.com/<你的用户名>/agent-rank-dashboard.git
git push -u origin main

# 之后 Settings → Pages → Deploy from main / root → Save
```
